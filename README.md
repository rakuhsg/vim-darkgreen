# vim-darkgreen

## Installation

Install package with your plugin manager.

```
Plug 'rakuhsg/vim-darkgreen'
```

Set color scheme to `darkgreen`

```
colorscheme darkgreen
```

## Building a theme

### Requirements

- [vim-colortemplate](https://github.com/lifepillar/vim-colortemplate)

```
:Colortemplate ~/.vim
```
